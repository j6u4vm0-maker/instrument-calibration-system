# PrecisionTrack | 儀器量測校證系統

這是一個獨立於 LabNexus 的專業儀器量測與校證管理系統。

## 核心設計理念
- **精確性**：嚴格追蹤校證週期與認證證書。
- **現代化感官**：採用深色模式、玻璃擬態（Glassmorphism）與 Outfit 字體。
- **易用性**：直觀的側選單與頂部全局搜尋。
- **CRUD 完整性**：凡涉及新增輸入資料的功能，必須同步規劃「修改」與「刪除」功能。


## 快速開始

1. 開啟終端機。
2. 進入專案目錄：
   ```bash
   cd instrument-calibration-system
   ```
3. 安裝依賴（如果尚未安裝）：
   ```bash
   npm install
   ```
4. 啟動開發伺服器：
   ```bash
   npm run dev
   ```
5. 在瀏覽器開啟 `http://localhost:3000`。

## 主要功能模組
- **儀器概覽**：即時統計與狀態監控。
- **校證計畫**：自動化提醒即將到期的設備。
- **紀錄管理**：電子化存檔校證報告與歷史。
- **系統設定**：管理權限與儀器類別。
